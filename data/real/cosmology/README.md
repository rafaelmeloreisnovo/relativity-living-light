# data/real/cosmology

Status: **Parcial real em preparação**.

## Escopo
Cadeia de custódia para datasets cosmológicos usados na validação da camada RLL.

## Fontes previstas
- DESI DR2 (BAO).
- Pantheon+ (SNe Ia).
- Planck 2018 (chains/likelihoods).
- H(z) cosmic chronometers.
- fσ8 (crescimento de estrutura).

## Registro obrigatório por dataset
- Fonte oficial e versão;
- URL/DOI;
- Licença/termos de uso;
- Data de acesso (UTC);
- Hash SHA-256 do dado bruto;
- Método de download;
- Script de ingestão/normalização;
- Observações sobre covariâncias e recortes aplicados.

## Limitações estatísticas a documentar
- Dependência de escolhas de prior e parametrização.
- Degenerescências entre parâmetros cosmológicos.
- Sensibilidade a seleção de subconjuntos de dados.
- Correlações/covariâncias incompletas ou heterogêneas entre sondas.
- Diferenças de calibração sistemática entre catálogos.
