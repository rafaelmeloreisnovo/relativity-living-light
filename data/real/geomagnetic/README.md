# data/real/geomagnetic

Este diretório é reservado para dados reais relacionados à AMAS/SAA.

## Política de ingestão
Somente inserir dados quando houver, de forma explícita:
- fonte primária/institucional;
- URL/DOI;
- data de origem;
- data de acesso;
- versão;
- licença de uso;
- hash do arquivo baixado (quando aplicável);
- script/processo de transformação;
- saída derivada e limitações.

Sem esses metadados, o repositório deve manter o status: **Parcial real em preparação**.
