# Editorial Safety Gate

## Status

`editorial_gate / respectful_use / claim_boundary`

## Purpose

This note defines what may safely enter from the complementary texts.

Only the coherent, constructive and respectful layer should be reused.

The goal is to preserve useful bridges without creating unnecessary conflict with religious, philosophical, cultural or scientific traditions.

## Allowed use

Use the material for:

```text
epistemic framing;
careful analogy;
translation awareness;
methodological humility;
respectful comparison;
claim-boundary discipline;
continuity with future repositories.
```

## Blocked use

Do not use the material for:

```text
doctrinal judgment;
sectarian dispute;
attack on any tradition;
claim of religious superiority;
claim of scientific validation;
replacement of evidence by metaphor;
confusion between theology, philosophy and empirical proof.
```

## Good-part rule

Keep only the good part of the situation:

```text
what helps clarify;
what helps respect;
what helps preserve continuity;
what helps separate regimes;
what helps avoid inflation;
what helps future work start with context instead of emptiness.
```

If a passage can be read as attacking a tradition, overstating theology or promoting a scientific claim without evidence, it should be softened, moved to a more appropriate repository, or left out.

## Transfer rule

For scientific repositories:

```text
use as analogy and epistemic framing only.
```

For theology, philosophy or language repositories:

```text
expand as interpretation, with respect for traditions and sources.
```

For mechanics or formula repositories:

```text
formalize mechanical parts with equations, tests and assumptions.
```

## Final boundary

A bridge is useful only while it remains kind, precise and bounded.

The purpose is not to win a tradition war.
The purpose is to preserve a good path of thought without inflating it into proof.
