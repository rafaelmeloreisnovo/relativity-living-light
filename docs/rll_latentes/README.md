## 1. Finalidade

Este diretório registra um método de investigação para elementos latentes, rejeitados, abortados, ignorados, subponderados ou óbvios demais para serem modelados.

A regra principal é:

> Uma analogia só sobe de poesia para método quando gera variável mensurável, invariante, falsificador ou reconstrução verificável.

Assim, uma relação não é aceita por beleza simbólica; ela precisa passar por coerência, integridade, prova e limite.

## 2. Fórmula de triagem

```math
S_L = \frac{C \times I \times P \times E \times R_c}{1 + R_u + A_m + V_b}
```

Onde:

- `C` = coerência entre domínios;
- `I` = integridade da fonte ou estrutura;
- `P` = prova mensurável ou reconstruível;
- `E` = ética do método, evitando simplificação destrutiva;
- `R_c` = recorrência em contextos distintos;
- `R_u` = ruído;
- `A_m` = ambiguidade;
- `V_b` = viés de base, escola, linguagem, instrumento ou estatística.

Interpretação:

- `0.00–0.25`: metáfora fraca;
- `0.25–0.50`: ponte possível;
- `0.50–0.75`: hipótese estruturável;
- `0.75–0.90`: modelo operacional;
- `0.90–1.00`: núcleo canônico, desde que tenha falsificador forte.

## 3. Geometria operacional

O índice usa geometria como linguagem de verificação, não como ornamento:

- polinômio: expressão algébrica de uma curva ou campo;
- quadrado: estabilização por área, incluindo completar quadrados;
- Pitágoras: ponte entre deslocamento, área, energia e erro residual;
- triângulo equilátero: simetria e estabilidade pela altura `sqrt(3)/2`;
- triângulo reto: decomposição ortogonal e medição;
- triângulo isósceles: eixo comparativo entre emissor e receptor;
- círculo interno: continuidade local;
- toro: recorrência com fase.

Resumo:

```math
polinômio \oplus Pitágoras \oplus triângulo \oplus círculo \oplus toro = campo de reconstrução
```

Nenhuma informação chega pura. Ela atravessa emissor, meio, distorção, instrumento, receptor, observador e interpretação.

Um reflexo no para-brisa, por exemplo, depende de luz, vidro, ângulo, olho, movimento, fonte, umidade, temperatura, vibração, índice de refração, câmera, GPS, magnetismo, pressão, radiação, gravidade e taxa de amostragem.

```math
I_{obs}=f(E,R,O,M,\theta,n,T,\mu,H,P,B,g,Hz)
```

Onde `E` é emissor, `R` receptor, `O` observador, `M` meio, `n` índice de refração, `T` temperatura, `mu` viscosidade, `H` umidade, `P` pressão, `B` campo magnético, `g` gravidade e `Hz` frequência/taxa de amostragem.

## 5. Frações, bases e fricção numérica

Igualdade aritmética não garante igualdade informacional.

Exemplo:

```math
77/33 = 7/3
```

Após simplificação, os valores coincidem. Porém `77/33` pode carregar genealogia simbólica, base, contagem ou unidade diferente de `7/3`.

Princípio:

```math
igualdade_numérica \neq igualdade_informacional
```

## 6. Fadiga estrutural e margem

A parábola técnica da asa do Concorde entra como regra de método:

- um sistema pode funcionar por muito tempo e ainda falhar por ciclos acumulados;
- pousos, decolagens, cisalhamento, torção e eventos raros revelam fragilidades;
- estatística que despreza caudas pode quebrar uma teoria, uma asa ou uma ponte conceitual.

```math
segurança \neq ausência_de_falha_observada
```

Segurança real exige margem, ciclo, ruído extremo, evento raro e teste de fadiga.

A_{aprendizado}=C_u \times D_i \times T_e \times R_p - R_n - Q_c

## 8. Complexidade e N≠NP

A explosão combinatória indica que verificar uma ligação local pode ser muito mais simples que provar o mapa total.

```math
verificar_uma_ligação \ll provar_todas_as_ligações_possíveis
```

O índice aceita sementes parciais, mas não finge completude.
