# RLL Cosmology Postdoctoral Synthesis (Formal Technical Draft)

## Executive Position
This document frames **Relativity Living Light (RLL)** as a **testable effective cosmological extension**, not a finalized fundamental theory. The core requirement is explicit comparability against ΛCDM and canonical dynamic-dark-energy baselines under reproducible statistical criteria.

## Formal model statement
\[
E^2(a)=\Omega_r a^{-4}+\Omega_m a^{-3}+\Omega_\Lambda+\Omega_{s0}[f(a)+(1-f)a^{-3}]+\Omega_{B0}a^{-4}+\Omega_{P0}a^{-4}
\]
with
\[
f(z)=\frac{1}{1+\exp((z-z_t)/w_t)},\quad z=a^{-1}-1.
\]

### Limiting-case constraint
\[
\lim_{\Omega_{s0},\Omega_{B0},\Omega_{P0}\to 0}E^2_{RLL}(a)=E^2_{\Lambda CDM}(a).
\]

This constraint is non-negotiable for scientific comparability.

## Validation hierarchy
1. **Expansion block**: DESI BAO + Pantheon+ + H(z).
2. **Early-universe consistency**: Planck + ACT priors.
3. **Growth block**: fσ8/lensing (BOSS/eBOSS; Euclid/Rubin as data mature).
4. **Stress/anomaly robustness**: extreme-system residual penalties (e.g., runaway SMBH class).

## Inference and model selection
- Likelihood assembly: \(\mathcal{L}=\mathcal{L}_{BAO}\,\mathcal{L}_{SNe}\,\mathcal{L}_{CMB}\,\mathcal{L}_{growth}\).
- Ranking metrics: χ², AIC, BIC, posterior predictive checks.
- Interpretation policy: no confirmation claims prior to cross-block stability.

## Professional claim boundaries
**Allowed now**
- RLL is a testable effective extension.
- RLL contains ΛCDM as a limit.
- RLL can be evaluated with standard cosmological probes.

**Forbidden until full validation**
