# Media Fields Map

Status: exploratory.
Purpose: map physical media into measurable fields.

## Fields

- density_rho: medium density.
- temperature_T: thermal state.
- pressure_P: compressive state.
- viscosity_eta: transport resistance.
- magnetic_B: magnetic field.
- opacity_kappa: absorption/scattering.
- ion_fraction_chi: ionized component.

Rule: each field needs unit, source, uncertainty and observable target.
