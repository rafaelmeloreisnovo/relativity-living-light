## Finalidade

Este índice reúne casos observacionais úteis ao RLL/MCRP como trilha de leitura, curadoria e validação cruzada. O objetivo não é transformar notícias em prova, mas preservar sinais relevantes com classificação epistemológica clara:

1. **divulgação científica** — notícia, release, resumo institucional;
2. **registro observacional** — objeto, evento, levantamento, instrumento, série temporal;
3. **literatura primária** — paper, preprint técnico, catálogo ou base de dados;
4. **uso RLL** — hipótese de comparação, caso didático, validação ou limite de alegação.

## Regra de promoção para `docs/REFERENCES.md`

Um registro só deve virar referência canônica `EXT-*` quando houver:

- URL ou DOI estável;
- autor/instituição/veículo identificado;
- ano/data de publicação;
- separação entre fonte jornalística e literatura primária;
- indicação explícita de uso: contexto, validação, refutação, limite ou comparação.
