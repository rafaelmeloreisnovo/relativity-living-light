# Epistemic Analogies Registry

## Status

`governance_record / epistemic_analogy / claim_boundary`

## Purpose

This directory preserves essays that use fable, theology, philosophy, engineering and comparative symbolism as analogical tools for reasoning about evidence states.

These texts are not scientific validation artifacts.

They illuminate how to read claims without collapsing them into a poor binary:

```text
proved / not proved
true / false
believe / disbelieve
```

The operational target is the repository evidence-state discipline:

```text
VERIFIED
DECLARED_BY_AUTHOR
TOKEN_VAZIO
CONTRADICTION
```

## Claim boundary

Metaphor, theology, myth, fable, philosophy and parable may illuminate classification, intent, failure modes and epistemic caution.

They do not validate RLL, do not refute RLL, do not select a cosmological model, do not prove a coefficient, and do not replace data, prediction, calculation, baseline comparison, uncertainty/covariance and falsification.

## Use rule

Use these essays as interpretive scaffolding only:

```text
analogy -> classification discipline
classification discipline -> better audit language
better audit language -> safer scientific workflow
```

Never use them as:

```text
analogy -> scientific validation
parable -> physical proof
symbol -> numerical result
```

## Editorial policy

The texts should preserve theological and philosophical seriousness while staying operationally modest.

A good essay in this directory must:

1. mark analogy as analogy;
2. separate literary illumination from empirical proof;
3. avoid claim inflation;
4. preserve the four-state evidence discipline;
5. connect back to falsifiable workflows, not replace them.
