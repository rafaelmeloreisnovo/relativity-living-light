# B) Modelo cosmológico fechado (definição + condições)

## Fechamento
Assuma k=0 e imponha Ω_total=1 no presente:
Ω_m + Ω_r + Ω_Λ + Ω_s0 = 1.

## Radiação completa
Ω_r = Ω_γ + Ω_ν(rel) com N_eff, e transição para neutrinos massivos Ω_ν(a) se você quiser z alto.

## Termo de superposição (forma fechada sugerida)
g(a)= f_ext(a) a^{-n_ext} + (1-f_ext(a)) a^{-n_col}
f_ext(a)= 1 / (1 + (a/a_t)^p ),  a_t=1/(1+z_t)

H^2/H0^2 = Ω_m a^{-3} + Ω_r a^{-4} + Ω_Λ + Ω_s0 g(a)

## Condições
- positividade: ρ_total(a) ≥ 0
- estabilidade: c_s^2 ≥ 0, sem ghosts na EFT
- early universe: Ω_sup(a)<<Ω_r em z~10^3 (ou justificar)
